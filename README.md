# NEON BEAT

ネオンの世界観で2種類のゲームを遊べる、ブラウザ向けリズムゲームです。

## LEVEL 1 — NEON BEAT

1. `START GAME`を押す
2. ノーツが判定ラインに重なる瞬間にSpaceキー、または画面の`TAP`を押す
3. 30秒間でハイスコアを目指す

長短を織り交ぜたリズムパターンでノーツが流れます。音源ファイルは使用せず、Web Audio APIで音を生成しています。

## LEVEL 2 — NEON ECHO

1. `LISTEN`中にシアンとピンクの光・音・間隔を覚える
2. `REPEAT`に変わったら、同じ順番とリズムをF・Jキーまたは左右のパッドで再現する
3. 全4ラウンドを進み、ハイスコアを目指す

ラウンドごとに4音、6音、8音、10音と長くなり、テンポも少しずつ上がります。

## ローカルで起動

```sh
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開いてください。

## 構成

- `style.css` — レスポンシブなネオンデザイン
- `game.js` — 音声、ノーツ、判定、スコア処理
- `SPEC.md` — ゲーム仕様書
